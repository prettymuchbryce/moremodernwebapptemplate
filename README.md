###More Modern Web App Template

This is currently my go-to starter template for spinning up web stuff.

####included
* ES6
* Express
* Bootstrap
* React
* react-router
* redux
* LESS
* lodash
* webpack
* bluebird
* EJS
* superagent

### install deps
    npm install

### compile frontend
    npm run build

### start server
    npm start

### watch
    npm run watch
