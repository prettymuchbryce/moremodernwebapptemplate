###More Modern Web App Template

####included
* ES6
* Express
* Bootstrap
* React
* react-router
* redux
* LESS
* lodash
* webpack
* EJS
* superagent

####installation

### install some global stuff if you don't have it
    npm install webpack -g

### install deps
    npm install

### compile frontend
    npm run build

### start server
    npm start

### watch
    npm run watch
